## Getting Started

Clone the repo and run `streamlit run streamlist_app.py`

### Installation

Install streamlit and python packages first

- pip
  ```sh
  git clone git@github.com:mtreilly/constructioncost.git
  cd constructioncost
  pipenv install
  ```
